# Ecom-Redux #

## What it is: ##

This application allows users to browse items in an online store and add to their cart for purchase. It uses React-Redux and MongoDB to display and store information.

## How to use: ##

When visiting the site, the user will be able to create an account, which will automatically log them in. The main page displays all available products, which can be added to the cart
